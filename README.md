# confusingfetch
a fetch tool thats.. confused?

## Installation

1. Download the (less)confetch file or copy it to your own
2. chmod +x (less)confetch
3. optional, mv (less)confetch /usr/local/bin/

## usage

./(less)confetch
<br>
or if you did step 3
<br>
(less)confetch

## flags

-v, --version
-h, --help

-l, --logo

-a, --accent

-k, --key

-K, --key2

-L, --larp

-C, --confused

-N, --not-confused
